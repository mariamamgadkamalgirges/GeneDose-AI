# AI Prediction Model

GeneDose AI uses an evidence-based clinical decision support model to assist oncologists in selecting personalized therapies for AML patients.

## Input Data

- Patient Information
- NGS Results
- Detected Gene Mutations
- ELN Risk Classification
- Clinical Guidelines

## AI Analysis

The AI engine evaluates genomic alterations and maps them to evidence-based therapeutic options.

Examples include:

- FLT3 → Midostaurin
- IDH1 → Ivosidenib
- IDH2 → Enasidenib

## Output

- Risk Classification
- Personalized Treatment Recommendation
- Clinical Report
